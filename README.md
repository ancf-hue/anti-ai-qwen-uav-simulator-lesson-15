##  О проекте

**«Урок 15: Наставник»** — интерактивный тренажёр, где вы обучаете AI-ученика управлять FPV-дроном. Ученик не может справиться без вашей помощи!

### 🎮 Геймплей

Вы — наставник. Ваша задача:
- 💡 **Давать подсказки** — показывать путь к цели (8 сек)
- 🎮 **Перехватывать управление** — лететь самому (5 сек)  
- 👍 **Подбадривать** — повышать уверенность ученика (+20%)

Без ваших действий ученик **НЕ справится**!
